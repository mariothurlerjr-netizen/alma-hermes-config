# Composio app priorities

Use when reactivating or scoping Composio integrations in Hermes.

## Order of activation
1. Gmail
2. Notion
3. Supabase
4. Outlook
5. LinkedIn
6. Instagram
7. All Grow, only if the use case is clear

## Why this order
- Gmail, Notion, Supabase, Outlook cover daily operator work and low-ambiguity retrieval.
- LinkedIn is useful as a job filter, not as a generic always-on source.
- Instagram is useful for campaign monitoring, not as a first-priority operational surface.
- All Grow is optional and can wait until the higher-signal apps are stable.

## Scoping rule
Start with a filtered toolkit, do not expose every connected app at once if that will inflate context or make tool choice noisy.
