# Agent cockpit canonical format

Use this when Mario asks for today’s goals, priorities, or status by agent.

## Default shape
- agent
- today’s goal
- success metric
- main blocker
- pause condition

## Delivery rules
- Keep it short and operational.
- If asked for multiple agents, return one block per agent.
- Prefer explicit quotas or targets over vague descriptions.
- If a quota is unclear, state the assumption and move on with the closest operational estimate.

## Example
- ORION, 100 leads/day, success is qualified leads, blocker is data quality, pause if lead quality drops below threshold.
